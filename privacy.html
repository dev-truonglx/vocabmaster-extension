VocabMaster Privacy Policy

Last updated: 2024-01-20

1. Information We Collect

  • Learning progress and statistics
  • Vocabulary lists and saved words
  • User preferences and settings

2. How We Use Your Information

  • To save your learning progress
  • To synchronize your vocabulary across devices
  • To improve the extension's functionality

3. Data Storage

All data is stored locally in your browser using Chrome Storage API. We use:

  • Chrome Storage Sync for settings and preferences
  • Chrome Storage Local for vocabulary and progress data

4. Third-party Services

We use the following third-party services:

  • Google Translate API for translation features
  • Chrome Storage API for data synchronization

5. Data Security

We implement security measures to protect your data:

  • Local storage encryption
  • Secure API communications
  • Regular security updates

6. Contact

For privacy concerns, contact us at: privacy@vocabmaster-extension.com
